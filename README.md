# claude-skills-franco

Skills personales de Claude Code para sincronizar entre PCs.

## Skills incluidas

- sistemafrd-context: contexto completo del proyecto SistemaFRD

## Instalacion en PC nueva

1. git clone este repo a Documents/
2. Ejecutar: .\install.ps1

## Workflow

Cuando modifiques una skill en ~/.claude/skills/, copiala al repo y haz commit + push.
En otra PC: git pull + .\install.ps1
